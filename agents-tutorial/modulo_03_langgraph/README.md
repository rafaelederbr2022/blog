# Module 3: LangGraph

## Description

This module covers building stateful workflows with LangGraph. You will learn how to create execution flows with state management, conditional and dynamic routing, human intervention points, checkpointing, and long-running workflows.

## Exercises

| # | Exercise | Description |
|---|----------|-------------|
| 15 | Basic StateGraph | Creating and executing a simple graph with nodes and edges |
| 16 | State Management | Manipulating and persisting state between nodes |
| 17 | Conditional Routing | Routing based on state conditions |
| 18 | Dynamic Routing | Dynamic routing based on LLM output |
| 19 | Human-in-the-Loop | Human intervention points in the workflow |
| 20 | Checkpointing | Saving and restoring graph state |
| 21 | Approval Workflow | Flows that pause for human approval |
| 22 | Long Running Workflows | Executing long-duration flows with persistence |

## Prerequisites

- Module 1 (LangChain Fundamentals) completed
- Module 2 (RAG) recommended for context
- Understanding of state machines and graph concepts

## Key Concepts

- StateGraph creation and execution
- State definition with TypedDict
- Node functions and edge connections
- Conditional edges and routing logic
- Human-in-the-loop patterns
- State checkpointing and persistence
- Workflow interruption and resumption
- Long-running process management
